# Riichi Mahjong Telegram Bot

Данный проект представляет собой Telegram-бота, разработанного для сообщества игроков в **Риичи-маджонг**. Бот служит инструментом для координации игроков, поиска партнеров в разных городах и быстрого доступа к обучающим материалам.

## Основные функции

### 1. Координация игроков по городам
Бот позволяет пользователям регистрировать свое местоположение, что помогает организовывать офлайн-встречи и игры.
- `/my_city [город]` — Установить или изменить свой город.
- `/cities` — Просмотреть список всех городов, в которых есть зарегистрированные игроки.
- `/users_from_city [город]` — Получить список игроков в конкретном городе с активными ссылками на их профили.
- `/leave_city` — Удалить себя из базы данных городов.

### 2. Инструменты для игроков
- `/hand [код_руки]` — Генерирует изображение маджонговой руки (используя API tempai.net). Поддерживает различные форматы, включая закрытые тайлы, повернутые тайлы и ака-доры.
- `/links` — Выводит интерактивное меню с полезными ссылками: игровые платформы, учебные материалы, инструменты статистики и ссылки на сообщества.

## Технические детали

- **Язык**: Python 3.9+
- **Библиотека**: `python-telegram-bot` (v13.13)
- **Хранение данных**:
    - `cities_mahjong/` и `cities_test/` — контекстные файлы городов с игроками, клубами и рейтингами.
    - `links.json` — файл с полезными ссылками.
- **Безопасность**: Работа бота ограничена списком разрешенных ID чатов (`ALLOWED_CHATS`), что предотвращает использование бота в посторонних группах.

## Как это используется
Проект активно используется в русскоязычном мадзянговом комьюнити для социализации игроков, упрощения процесса сбора на игры и как быстрый справочник ресурсов прямо в мессенджере.

## Запуск

Локально:

```powershell
poetry env use python
poetry lock
poetry install
poetry run python main.py
```

На сервере:

```bash
python3 -m pip install --user poetry
poetry install
poetry run python main.py
```

Если `poetry.lock` уже актуален и зависимости не менялись, обычно достаточно:

```bash
poetry install
poetry run python main.py
```
