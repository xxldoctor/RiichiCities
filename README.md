# Riichi Mahjong Telegram Bot

Данный проект представляет собой Telegram-бота, разработанного для сообщества игроков в **Риичи-мадзянг**. Бот служит инструментом для координации игроков, поиска партнеров в разных городах и быстрого доступа к обучающим материалам.

## Основные функции

### 1. Координация игроков по городам
Бот позволяет пользователям регистрировать свое местоположение, что помогает организовывать офлайн-встречи и игры.
- `/my_city [город]` — Установить или изменить свой город.
- `/cities` — Просмотреть список всех городов, в которых есть зарегистрированные игроки.
- `/users_from_city [город]` — Получить список игроков в конкретном городе с активными ссылками на их профили.
- `/leave_city` — Удалить себя из базы данных городов.

### 2. Инструменты для игроков
- `/hand [код_руки]` — Генерирует изображение мадзянговой руки (используя API tempai.net). Поддерживает различные форматы, включая закрытые тайлы, повернутые тайлы и ака-доры.
- `/links` — Выводит интерактивное меню с полезными ссылками: игровые платформы (Tenhou, Mahjong Soul), учебные материалы, инструменты статистики и ссылки на сообщества.

### 3. Администрирование
Бот включает систему прав доступа (проверка на администратора чата или по списку `ADMIN_USERNAME`), позволяющую:
- Переименовывать города (`/rename_city`).
- Удалять пользователей (`/remove_user`) или целые города (`/remove_city`).
- Использовать отладочные команды (`/debug`, `/debug_all`).

## Технические детали

- **Язык**: Python 3.9+
- **Библиотека**: `python-telegram-bot` (v13.13)
- **Хранение данных**:
    - `city_users.json` — база данных городов и ID пользователей.
    - `links.json` / `linksn.json` — файлы с полезными ссылками.
- **Безопасность**: Работа бота ограничена списком разрешенных ID чатов (`ALLOWED_CHATS`), что предотвращает использование бота в посторонних группах.

## Как это используется
Проект активно используется в русскоязычном мадзянговом комьюнити для социализации игроков, упрощения процесса сбора на игры и как быстрый справочник ресурсов прямо в мессенджере.
